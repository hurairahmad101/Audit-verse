# Rebrand to AuditVerse.AI & Hide ComplyChat

## What & Why
Rename the product from "ComplyVerse" to "AuditVerse.AI" across the entire application, and remove the ComplyChat navigation link from the sidebar so users no longer see or can access that module.

## Done looks like
- Login and register pages show "AuditVerse.AI" as the product name
- Browser tab title reads "AuditVerse.AI"
- PDF reports generated by the audit reporting module reference "AuditVerse.AI"
- ComplyChat no longer appears in the sidebar navigation
- All other functionality is unchanged

## Out of scope
- Removing ComplyChat backend routes or files (just hide the UI link)
- Changing any logos or color themes

## Steps
1. **Replace product name in frontend** — Update "ComplyVerse" to "AuditVerse.AI" in the layout metadata, login page, register page, and the audit reporting page footer/header text.
2. **Replace product name in backend** — Update "ComplyVerse" to "AuditVerse.AI" in the PDF report generation code (reporting.py) in all places it appears.
3. **Hide ComplyChat from sidebar** — Remove the ComplyChat nav entry from the sidebar navigation array so it no longer renders in the UI.
